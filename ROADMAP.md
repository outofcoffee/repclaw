# Roadmap

## Features

- Model selector
- Effort/thinking selector for model
- Command execution mode - run command using `! some command` on gateway host
- Use slash commands instead of ctrl+c to exit
- Support locally installed agent skills
- Show sessions stats, such as context usage
- Create new agent from agent selector
- Auto select `main` agent if only single agent configured
